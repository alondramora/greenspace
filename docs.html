Get Space by ID

To get a green space by it's MongoDB ID, use the following code in a GET request:

http://localhost:3001/spaces/:id

JSON

	"_id": "64ab3f9b2b50806d8e80c200",

Update or Delete a Space

To update a space, using the following code in a PUT request. to delete a space, use the following code in a DELETE request.

http://localhost:3001/spaces/:id
